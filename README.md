### Prerequisities

- Install Docker & Docker Compose

<br>

### Usage

- Download and navigate to project
```
git clone https://github.com/Jonathan-Eid/hrdashboard.git
cd hrdashboard
```

- Start project
```
docker-compose build
docker-compose up
```

- Navigate to ```localhost:5173```
