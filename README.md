### Prerequisities

- Install [Docker & Docker Compose](https://docs.docker.com/get-docker/)
<br>

### Usage

- Download and navigate to project
```
git clone https://github.com/Jonathan-Eid/hrdashboard.git
cd hrdashboard
```

- Start project
```
docker-compose up -d
```

- Navigate to ```localhost:5173```

### Teardown

```
docker-compose down
cd ..
sudo rm -rf hrdashboard
```
